== Installation ==

1. Upload the pms-add-on-tax folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A new Tax tab will be added under PMS -> Settings page from where you can configure the tax rates
