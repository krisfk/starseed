Drips your content. Different users get access to different amounts of content based on how long they've been an active member.

= Website =

http://www.cozmoslabs.com/wordpress-paid-member-subscriptions/

== Installation ==

1. Upload the pms-add-on-content-dripping folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a new page and use one of the shortcodes available. Publish the page and you're done!
