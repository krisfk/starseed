= Website =

http://www.cozmoslabs.com/wordpress-paid-member-subscriptions/

== Installation ==

1. Upload the pms-add-on-pay-what-you-want folder to the '/wp-content/plugins/' directory
2. Activate the add-on through the 'Plugins' menu in WordPress
3. When adding or editing a subscription plan you can configure the pay what you want options
